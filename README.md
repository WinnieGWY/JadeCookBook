# Jade小书

最近学习Jade，搜索了一下中文教程，却发现真的很少(几乎没有...)。好不容易找了一篇博客，发现还是比较早的博客而且只是把官方文档的例子搬过来了而已。所以就打算写一本Jade小书，既算是学习笔记，也算为Jade在天朝的发展做一点贡献吧。

其实Jade官方已经更名为Pug了，但是现在并没有流行开，至少在国内是这样的。所以本书还以Jade为例，在后边会介绍一下Pug的新特性。
